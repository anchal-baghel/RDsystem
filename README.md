# 💰 RDsystem - Recurring Deposit Management

A professional **Full-Stack Finance Application** designed to streamline the management of recurring deposits, automate fine calculations, and maintain digital passbooks.

---

## 🚀 Key Features

* **Comprehensive Dashboard:** Real-time view of all RD holders with KYC (PAN/Aadhaar) and Nominee details.
* **Full CRUD Operations:** Seamlessly **Add, View, and Update** user profiles and investment plans.
* **Smart Passbook Entry:** Automated system to log monthly installments.
* **Advanced Fine Logic:** Automatically calculates **'Late Days'** and **'Fine Amount'** based on payment dates.
* **Responsive UI:** Fully optimized for both desktop and mobile viewing.
